# Power BI - KPI's
Dashboard operacional desenvolvido em Power BI para análise de produção e desempenho no setor de E-commerce.

Este projeto consiste no desenvolvimento de um Dashboard Operacional de E-commerce, com foco na análise de desempenho produtivo por funcionário, turno e data.

O relatório permite acompanhar indicadores como total de transações, itens coletados, média de produção e ranking de performance, auxiliando a identificação de gargalos operacionais e apoiando a tomada de decisão gerencial.

O projeto foi construído utilizando Power BI, com aplicação de DAX para criação de medidas analíticas e Power Query (Linguagem M) para tratamento e modelagem dos dados.

### Principais Indicadores (KPIs)
- Total de Transações Realizadas
- Total de Itens Coletados
- Média de Produção por Funcionário
- Ranking de Performance (Top 10)

### Ferramentas Utilizadas
- Power BI
- DAX
- Power Query (M)
